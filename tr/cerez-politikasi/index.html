Çerez Politikası

Çerezlerin Kullanımı ve Yönetimi

Makbaş LTD.ŞTİ., web sitemizde çerezler kullanarak kullanıcı deneyimini iyileştirmeyi hedefler. Bu çerez politikası, çerezlerin ne olduğu, nasıl kullanıldığı ve kullanıcıların çerezleri nasıl yönetebileceği hakkında bilgi sağlar.

Çerez Nedir?

Çerezler, web siteleri tarafından tarayıcınıza yerleştirilen küçük veri dosyalarıdır. Bu dosyalar, web sitelerinin tarayıcınızı tanımasına ve belirli bilgileri hatırlamasına yardımcı olur. Çerezler, web sitesi işlevselliğini artırmak ve kullanıcı deneyimini kişiselleştirmek için kullanılır.

Çerez Türleri

Web sitemizde kullanılan çerez türleri şunlardır:

  • Kesinlikle Gerekli Çerezler: Web sitesinin temel işlevlerini yerine getirmesi için gerekli olan çerezlerdir. Bu çerezler olmadan, web sitesinin bazı bölümleri çalışmayabilir.
  • Performans Çerezleri: Web sitemizi nasıl kullandığınızı analiz eden çerezlerdir. Bu çerezler, web sitemizin performansını artırmak amacıyla kullanıcı davranışlarını toplar.
  • Fonksiyonellik Çerezleri: Kullanıcı tercihlerini ve ayarlarını hatırlayan çerezlerdir. Bu çerezler, kullanıcı deneyimini kişiselleştirmek ve web sitesinin kullanıcı dostu olmasını sağlamak için kullanılır.
  • Reklam Çerezleri: İlginizi çekebilecek reklamları göstermek için kullanılan çerezlerdir. Bu çerezler, web tarayıcınızda gezinme alışkanlıklarınızı takip eder ve hedeflenmiş reklamlar sunar.

Çerezlerin Yönetimi

Tarayıcınızın ayarlarını değiştirerek çerezleri yönetebilir veya silebilirsiniz. Çerezleri devre dışı bırakmak, web sitemizin bazı işlevlerinin doğru çalışmamasına neden olabilir. Çerez ayarlarınızı tarayıcınızın gizlilik veya güvenlik ayarlarından yapabilirsiniz.

Çerez Politikamızın Güncellenmesi

Bu çerez politikası zaman zaman güncellenebilir. Herhangi bir değişiklik durumunda, güncellenmiş politika web sitemizde yayımlanacaktır. Çerez politikamızı düzenli olarak gözden geçirmeniz önerilir.
